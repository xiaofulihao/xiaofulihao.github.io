2.Markdown 换行

几乎每个 Markdown 应用程序都支持两个或多个空格进行换行
但这是有争议的，因为很难在编辑器中直接看到空格，
并且很多人在每个句子后面都会有意或无意地添加两个空格。
由于这个原因，你可能要使用除结尾空格以外的其它方式来换行。
幸运的是，几乎每个 Markdown 应用程序都支持另一种换行方式：HTML的标签

3.Markdown 引用

要创建块引用，请在段落前添加一个 > 符号

如：

引用测试

多个段落的块引用在句子前面添加多个 > 符号

如：

引用测试01

引用测试02
引用测试03

嵌套块引用块引用可以嵌套。在要嵌套的段落前添加一个 >> 符号

如：

嵌套引用测试

嵌套引用测试内嵌

4.Markdown 列表

列表分为有序和无序

有序列表:
要创建有序列表，请在每个列表项前添加数字并紧跟一个英文句点。
数字不必按数学顺序排列，但是列表应当以数字 1 起始。

如：
数字顺序排序

  1. 列表1
  2. 列表2
  3. 列表3
  4. 列表4

数字重复排序

  1. 列表1
  2. 列表2
  3. 列表3
  4. 列表4

数字无序排序

  1. 列表1
  2. 列表2
  3. 列表3
  4. 列表4

无序列表
要创建无序列表，请在每个列表项前面添加破折号 (-)、星号 (*) 或加号 (+)。缩进一个或多个列表项可创建嵌套列表。

如：

  • 列表一
  • 列表二
  • 列表三

    嵌套列表

    • 列表四

5.Markdown 代码语法

单词或短语表示为代码，请将其包裹在反引号 (`) 中.

如：

help是帮助指令、

如果你要表示为代码的单词或短语中包含一个或多个反引号，则可以通过将单词或短语包裹在双反引号(``)中。

如：

我需要 `help` 来帮助我

6.Markdown 链接

链接文本放在中括号内，链接地址放在后面的括号中，链接title可选。
超链接Markdown语法代码：[超链接显示名](超链接地址 "超链接title")

如：

给链接增加 Title
链接title是当鼠标悬停在链接上时会出现的文字，这个title是可选的，它放在圆括号中链接地址后面，跟链接地址之间以空格分隔。

如：

这是一个链接 Markdown语法

使用尖括号可以很方便地把URL或者email地址变成可点击的链接

如；

<xiaofulihao.github.io>
fake@qq.com

引用链接具体语法见Markdiwn基础语法之Markdown链接

7.Markdown 图片

要添加图像，请使用感叹号 (!), 然后在方括号增加替代文本，图片链接放在圆括号里，括号里的链接后可以增加一个可选的图片标题文本。
插入图片Markdown语法代码：![图片alt](图片链接 "图片title")。

如：

链接图片

代码[![图片alt](图片链接)](跳转链接)
